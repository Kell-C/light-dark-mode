Quando decidi mudar de carreira para o desenvolvimento web, percebi que poderia continuar criando experiências significativas — só que dessa vez, na internet. Resolvendo problemas - da sala de aula ao desenvolvimento web!

Enfrentar desafios diários era parte da minha rotina — desde captar a atenção dos alunos até criar métodos eficazes de ensino. Quando entrei no desenvolvimento web, essa habilidade se tornou essencial para aprender novas tecnologias e resolver problemas técnicos. Com o aprendizado de React e Node.js, percebi que cada bug é um problema a ser resolvido, similar aos desafios educacionais que enfrentava antes.

Projetos

Iniciando a carreira em desenvolvimento web após anos de dedicação ao ensino, trago uma abordagem inovadora e focada em resolver problemas reais com tecnologia. Estou em transição para me tornar uma desenvolvedora web completa, com projetos que refletem minha paixão por codificação e ensino.

Quote Generator

Este projeto é um gerador de citações que exibe aleatoriamente uma de cerca de 300 citações inspiradoras. As citações são fornecidas por uma API personalizada, que contém uma coleção variada de autores e temas. O objetivo é criar uma ferramenta simples e acessível para gerar inspiração rapidamente.

kell-c.github.io/quote-generator/

E-commerce: Vitrine Web

Vitrine Web é um e-commerce desenvolvido com a stack MEAN (MongoDB, Express, Angular e Node.js), visando oferecer uma plataforma intuitiva para lojistas e consumidores. O projeto facilita o gerenciamento de produtos e proporciona uma experiência de compra simplificada, com uma interface responsiva e amigável. a aplicação inclui funcionalidades como busca de produtos, carrinho dinâmico, sistema de autenticação e painel administrativo para gerenciamento de estoque e pedidos.

kell-c.github.io/vitrineweb/

Infinite Scroll

Este projeto é um web app de microblogging de fotos, inspirado no Instagram, com infinite scroll. As imagens são carregadas continuamente enquanto o usuário navega pela página, proporcionando uma experiência fluida e ininterrupta. O conteúdo é alimentado dinamicamente pela Unsplash API, com acesso rápido e eficiente a um banco de imagens atual

kell-c.github.io/infinite-scroll/

Joke Teller

Joke Teller é um web app divertido que conta piadas automaticamente, usando APIs para buscar piadas e convertê-las em áudio. Ideal para uma boa risada a qualquer momento, o app permite que você receba piadas de maneira interativa e divertida.

kell-c.github.io/joke-teller/

Picture in Picture

Este projeto é um web app que permite configurar uma janela de vídeo como picture-in-picture (PiP) durante o compartilhamento. Com este recurso, o usuário pode minimizar o vídeo em uma pequena janela flutuante enquanto realiza outras atividades no navegador.
